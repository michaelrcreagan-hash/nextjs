## PART 1: ROTATION FRAMEWORK OVERVIEW

### The 4 Signals That Drive Rotation

| Signal | Type | Frequency | What It Triggers |
|---|---|---|---|
| **REGIME** | Macro | Daily | Broad risk-on/off, leverage |
| **ROTATE** | Seasonal/Quarterly | Quarterly | Sector/theme shifts |
| **TACTICAL** | Technical | Weekly | Entry timing within rotation |
| **EVENT** | Calendar-driven | Pre-event | Pre-FOMC, earnings, expiry hedges |

### Governing Principle

> **"Macro first, theme second, stock third, options fourth."**

Rotation priority:
1. **Regime** tells you HOW MUCH to invest
2. **Quarterly calendar** tells you WHERE to invest
3. **Relative strength + fund flows** tell you WHICH names within the sector
4. **ATR/IV signals** tell you HOW to structure the trade

## PART 6: THE 5-STAR ROTATION SIGNAL (Unified Entry)

Use this single composite signal to know **exactly** when to rotate:

```
ROTATION SIGNAL = RegimeScore + SeasonalScore + RS_Score + FlowScore + EventScore

RegimeScore:
├─ RISK-ON     = +3
├─ MIXED       = +1
├─ CAUTION     = -1
└─ RISK-OFF    = -3

SeasonalScore:
├─ Q1/Q4       = +2 (risk-on months)
├─ Q2/Q3       = -1 (neutral/defensive)
└─ Current month in seasonal top-5 = +1

RS_Score:
├─ Sector ETF > 50 DMA > 200 DMA = +2
├─ Sector ETF > QQQ YTD = +1
└─ Sector ETF below 200 DMA = -2

FlowScore:
├─ 3-month ETF inflow > $500M = +1
├─ 3-month ETF outflow > $500M = -1
└─ Neutral = 0

EventScore:
├─ Within 2 weeks of FOMC = -1
├─ After FOMC dovish pause = +2
├─ Quad witching week = -1
└─ Major earnings (NVDA) = -1 pre, +1 post beat

TOTAL SCORE:
├─ ≥+4 = AGGRESSIVE ROTATION INTO EQUITIES
├─ +2 to +3 = MODERATE ROTATION
├─ 0 to +1 = HOLD CURRENT
├─ -1 to -2 = DEFENSIVE ROTATION
└─ ≤-3 = CASH / BONDS ONLY
```

**Rotation Execution Rule:**
- Rotate **no more than once per quarter** unless triggered by regime shift
- Exit old sector **before** entering new sector (not simultaneous)
- Use 3-day VWAP as execution benchmark
