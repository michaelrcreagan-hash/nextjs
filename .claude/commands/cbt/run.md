<objective>
Execute a backtest run with the current strategy and configuration.
Save results as a tracked experiment. Support parameter overrides.
</objective>

<execution_context>
@strategies/{active}/strategy.py
@strategies/{active}/config.yaml
@strategies/{active}/.cbt/state.yaml
@strategies/{active}/backtest.py
</execution_context>

<process>

## 1. Parse Arguments

Extract parameter overrides from arguments:
- `--leverage=10` → Override leverage
- `--stop_loss=1.5` → Override stop loss %
- `--start_date=2024-01-01` → Override start date

## 2. Load Configuration

Read config.yaml and apply overrides.

## 3. Generate Experiment ID

```python
# Count existing experiments
existing = glob('experiments/exp_*.yaml')
next_num = len(existing) + 1
exp_id = f"exp_{next_num:03d}"
```

## 4. Execute Backtest

Run backtest.py:

```bash
cd strategies/{name} && python backtest.py --config config.yaml --output experiments/{exp_id}.yaml
```

Or execute inline if backtest.py supports it.

## 5. Calculate Metrics

Standard metrics:
- **total_return**: (final_equity - initial) / initial * 100
- **sharpe_ratio**: mean(returns) / std(returns) * sqrt(252)
- **sortino_ratio**: mean(returns) / downside_std * sqrt(252)
- **max_drawdown**: max peak-to-trough decline
- **win_rate**: winning_trades / total_trades * 100
- **profit_factor**: gross_profit / gross_loss
- **total_trades**: count of completed trades
- **avg_trade_duration**: mean holding period
- **avg_winner**: mean profit on winning trades
- **avg_loser**: mean loss on losing trades
- **largest_winner**: best single trade
- **largest_loser**: worst single trade
- **consecutive_wins**: longest winning streak
- **consecutive_losses**: longest losing streak

## 6. Save Experiment

Create `experiments/{exp_id}.yaml`:

```yaml
# Experiment: {exp_id}
# Strategy: {strategy_name}
# Generated by CBT Framework

  id: exp_003
  timestamp: "2026-02-01T22:30:00"
  parent: exp_002  # if iteration
  change: "Increased leverage from 5x to 10x"

parameters:
  account:
    initial_capital: 10000
  sizing:
    mode: percent
    percent_per_trade: 2.0
  leverage:
    default: 10  # overridden
  # ... full config snapshot

results:
  total_return: 34.5
  sharpe_ratio: 1.67
  sortino_ratio: 2.14
  max_drawdown: -15.2
  win_rate: 56.3
  profit_factor: 1.82
  total_trades: 143
  avg_trade_duration: "4h 23m"
  avg_winner: 1.23
  avg_loser: -0.89
  largest_winner: 5.67
  largest_loser: -3.21
  consecutive_wins: 8
  consecutive_losses: 5

prop_firm:  # only if prop_firm.enabled
  compliant: true
  max_dd_from_initial: -3.45
  daily_loss_breaches: 0
  target_reached: true
  target_bar: 1234
  breach_bar: null
  breach_reason: null

equity_curve_file: "trades/equity_{exp_id}.csv"
trades_file: "trades/trades_{exp_id}.csv"
```

Also save:
- `trades/trades_{exp_id}.csv` - Individual trade log
- `trades/equity_{exp_id}.csv` - Equity curve data

## 7. Compare to Previous

Compare against:
- Baseline (if exists)
- Previous run (if exists)
- Best run (if exists)

## 8. Update State

```yaml
experiments:
  count: 3
  current: exp_003
  best: exp_003  # update if new best
```
